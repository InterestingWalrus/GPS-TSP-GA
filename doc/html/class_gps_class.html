Detailed Description

Class to handle Waypoint Optimisation and Route generation for GPS Waypoints.

Class contains helper functions for GPS coordinate manipulation and code required to implement a genetic algorithm to solve the Travelling Salesman Problem. This is based on the solution presented by Daniel Shiffman in the Nature of Code Series on TSP as seen on https://thecodingtrain.com/CodingChallenges/035.1-tsp.html
